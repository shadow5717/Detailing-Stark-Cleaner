Backup & Restauración de Datos

Generar Backup

Crea una copia de seguridad de todos los datos del sistema. Se descargará un archivo JSON que puedes guardar en un lugar seguro.

Restaurar Sistema

Restaura todos los datos del sistema desde un archivo JSON de backup. Advertencia: Esta acción sobrescribirá todos los datos actuales.

Información del Backup

El sistema de backup guarda toda la información del sistema incluyendo:

  • Usuarios y permisos
  • Servicios de barbería y car wash
  • Productos del billar
  • Empleados registrados
  • Ventas e historial
  • Citas programadas
  • Vehículos registrados
  • Cierres de caja
  • Configuraciones del sistema

Recomendación: Realiza backups regularmente para proteger tu información.
